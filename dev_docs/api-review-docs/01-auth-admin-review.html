üí° Recommendations

  1. P1 Complete RBAC coverage - Add @Roles() to sessions.controller and ensure all admin endpoints are protected.
  2. P1 Standardize role names - Convert ADMIN, SUPER_ADMIN to lowercase (admin, super_admin) to match other services.
  3. P1 Add E2E tests - Test login, logout, refresh, password reset, user CRUD, role CRUD flows.
  4. P2 Add Swagger documentation - Add @ApiTags('Auth'), @ApiBearerAuth() to all controllers.
  5. P3 Reduce JWT expiry - Change access token expiry to 15 minutes for security.
