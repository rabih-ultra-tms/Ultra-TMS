üîê Access Control Model

Ticket Visibility Logic

Help Desk implements proper ownership-based access:

  • Ticket Creator: Can view, comment, close their own tickets
  • Assigned Agent: Can view, comment, update status, reassign
  • Support Manager: Can view all tickets in their category
  • Admin: Full access to all tickets
  • Internal Comments: Only visible to agents/admins, not ticket creator
