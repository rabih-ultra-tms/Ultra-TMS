## Freight Forwarding Features

### Ocean Freight

| Feature             | Description                |
| ------------------- | -------------------------- |
| FCL Booking         | Full container bookings    |
| LCL Consolidation   | Less than container        |
| Vessel Tracking     | Real-time vessel positions |
| B/L Management      | Draft, original, release   |
| Demurrage/Detention | Port charges               |

### Air Freight

| Feature          | Description       |
| ---------------- | ----------------- |
| AWB Management   | Master/house AWB  |
| Flight Booking   | Airline capacity  |
| Dimension/Weight | Chargeable weight |
| Special Handling | DG, perishables   |
| Track & Trace    | Flight tracking   |

### Customs

| Feature            | Description      |
| ------------------ | ---------------- |
| Entry Filing       | ABI integration  |
| Export Filing      | AES integration  |
| HTS Classification | Duty lookup      |
| Compliance         | C-TPAT, ISF      |
| Bond Management    | Continuous bonds |
