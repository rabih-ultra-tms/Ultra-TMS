## Description

## Key Design Considerations
- Real-time system health indicators with automatic escalation for critical infrastructure issues
- Multi-tenant overview with quick-access links to tenants experiencing issues or approaching usage limits

## Dependencies

- Tenant data for tenant overview metrics
- Infrastructure monitoring for system health
