## Description
Tracking interface for monitoring volume-based rebates earned from factoring companies, including rebate tiers, accrued amounts, payment history, and projected earnings.

## Key Design Considerations
- Rebate tier visualization showing current volume position relative to tier thresholds
- Projected rebate earnings calculator based on current factoring volume trends

## Dependencies
- Factor company rebate agreements from Factor Setup (Service 30, Screen 3)
- Purchase Requests (Service 30, Screen 4) for volume calculations
- Accounting service for rebate payment posting
