## Description
Live shipment tracking map showing real-time truck location, route visualization, origin/destination markers, estimated arrival time, and status timeline. Supports tracking by load number or BOL number.

## Key Design Considerations
- Real-time location updates with smooth map animation provide a premium tracking experience
- ETA calculation accuracy directly impacts customer satisfaction and planning

- Service 34 - ELD (ELD integration for real-time location)
