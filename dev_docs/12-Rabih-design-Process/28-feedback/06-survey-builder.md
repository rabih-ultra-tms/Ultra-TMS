## Description
Create and edit surveys with drag-and-drop question builder supporting multiple question types (multiple choice, rating, free text, matrix), conditional logic, and preview.

## Key Design Considerations
- Drag-and-drop question ordering with multiple question type templates
- Conditional logic builder for skip patterns and branching survey flows

## Dependencies
- Surveys list for survey management
- Question type library
- Preview/test infrastructure for survey testing before distribution
