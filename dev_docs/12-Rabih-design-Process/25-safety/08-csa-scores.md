## Description
FMCSA CSA (Compliance, Safety, Accountability) metrics dashboard showing all seven BASIC categories, percentile rankings, intervention thresholds, and historical trends.

## Key Design Considerations
- BASIC category gauges with clear threshold indicators (satisfactory, conditional, unsatisfactory)
- What-if analysis to project how new violations or inspections would affect scores

## Dependencies
- FMCSA SMS data integration
- Inspection and violation data
- Driver safety score calculations
