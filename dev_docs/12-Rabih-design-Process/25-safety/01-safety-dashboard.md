## Description
Safety overview dashboard showing key safety metrics, recent incidents, inspection results, driver safety trends, CSA score summaries, and upcoming safety-related deadlines.

## Key Design Considerations
- Critical safety alerts must be prominently displayed with urgency indicators
- KPI cards for incident rates, inspection pass rates, and CSA BASIC category scores

## Dependencies
- Incident tracking data
- Inspection records
- Driver safety scores
- CSA/FMCSA data integration
