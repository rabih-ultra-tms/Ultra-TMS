## Description
File a new safety incident report including incident type, date/time, location, description, involved parties, severity assessment, photos, and initial corrective actions.

## Key Design Considerations
- Must be mobile-friendly for drivers reporting incidents from the field
- Required fields should follow DOT/FMCSA reporting requirements

## Dependencies
- Driver service for driver lookup
- Equipment service for vehicle lookup
- Location/mapping service for incident location
- Document service for photo uploads
