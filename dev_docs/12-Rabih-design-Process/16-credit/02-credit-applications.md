## Description
Pending credit applications list showing application date, customer name, requested credit limit, D&B score, application status (pending, approved, denied), and assigned reviewer.

## Key Design Considerations
- Application queue with aging helps ensure timely processing (SLA tracking)
- Quick-view of D&B score and trade references enables rapid initial assessment
