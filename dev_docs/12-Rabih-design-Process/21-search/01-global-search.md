## Description
Full-text search across all entities in the system. Provides instant results as the user types, with categorized results spanning loads, customers, carriers, drivers, invoices, and more.

## Key Design Considerations
- Must provide near-instant results with typeahead/autocomplete behavior
- Results need to be categorized by entity type with clear visual distinction

## Dependencies
- All entity services for searchable data
- Authentication service for filtering results based on user permissions
