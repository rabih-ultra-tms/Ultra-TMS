## Screen Type

## Description
View a knowledge base article with rich content (text, images, videos, step-by-step instructions), related articles, helpfulness rating, and option to submit a ticket if the article didn't resolve the issue.

## Key Design Considerations
- Clean, readable article layout with table of contents for longer articles
- "Was this helpful?" feedback mechanism and escalation path to create a ticket

## Dependencies
- Knowledge Base list for navigation
- Content rendering engine for rich media
- New Ticket form for escalation from article
