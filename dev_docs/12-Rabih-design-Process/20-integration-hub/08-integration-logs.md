## Description
Activity log viewer showing all integration events including API calls, webhook deliveries, sync operations, errors, and data transformations with filtering and search capabilities.

## Key Design Considerations
- Advanced filtering by integration, event type, status, and date range with full-text search
- Expandable log entries showing request/response payloads for debugging integration issues
