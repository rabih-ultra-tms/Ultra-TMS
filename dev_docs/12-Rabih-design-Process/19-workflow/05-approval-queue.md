## Screen Type
List

## Description
Personal approval queue showing all pending approval requests assigned to the current user. Displays request type, requestor, summary, priority, requested date, and approve/deny action buttons.

## Key Design Considerations
- One-click approve/deny with optional comments maximizes approval throughput
- Mobile-friendly design is critical as approvers often work from phones

## Dependencies
- Workflow instances within this service
- All services that generate approval requests
