## Description
Market rate checking tool that queries external load boards for current spot rates on specific lanes, providing rate ranges, historical trends, and competitive positioning insights.

## Key Design Considerations
- Lane-based rate comparison across multiple data sources with confidence intervals
- Historical rate trend charts showing seasonal patterns and market movement for pricing decisions

## Dependencies
- Board Connections (Service 31, Screen 2) for data sources
- Rate Intelligence service for aggregated rate data
- TMS Core for lane and equipment type definitions
