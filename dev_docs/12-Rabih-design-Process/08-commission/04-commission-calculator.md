## Description
Commission calculation tool that processes completed loads against agent commission plans to generate commission amounts. Supports batch calculation for pay periods, adjustments, and override entries.

## Key Design Considerations
- Batch processing for monthly/bi-weekly pay periods with preview before finalizing
- Manual adjustment and override capability needed for special situations and corrections
