## Description
Compare rate sources side-by-side including internal rates, customer contract rates, carrier bids, and multiple external market data providers for the same lane.

## Key Design Considerations
- Side-by-side comparison layout with clear source attribution and data freshness
- Margin calculation showing profit/loss at different price points

## Dependencies
- Multiple rate data sources (internal, external, contract)
- Lane Rate Lookup for lane selection
- Customer and Carrier rate card data
