## Description
Financial reporting suite including Profit & Loss statement, Balance Sheet, Cash Flow statement, and Trial Balance. Supports date range selection, comparison periods, and export to PDF/Excel.

## Key Design Considerations
- Multiple report types with configurable date ranges and comparison periods add complexity
- Export capabilities (PDF, Excel) are essential for external stakeholder reporting

## Dependencies

- All financial data across the accounting service
