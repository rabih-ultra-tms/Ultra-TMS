# P1S-013: Agent Portal User Management

**Priority:** P1
**Service:** Agents
**Scope:** Admin interface for managing agent users and access

## Current State
No agent portal user management exists. The auth system supports roles but no agent-specific role or portal.

## Requirements
- List page for all agents
- Create/edit agent profiles
- Assign agent to customers/accounts
- Set commission rates per agent
- Enable/disable agent access
- Agent login credentials management

## Acceptance Criteria
- [ ] Agent list page with filtering
- [ ] Create agent with profile information
- [ ] Assign agent to customers
- [ ] Commission rate configuration
- [ ] Enable/disable agent
- [ ] Agent can log in with assigned credentials

## Dependencies
- Auth system role support for AGENT role
