# P1S-010: Contract Templates List

**Priority:** P1
**Service:** Contracts
**Scope:** List page for managing contract templates

## Current State
No contract management system exists in the frontend.

## Requirements
- List page showing all contract templates
- Filter by type (carrier agreement, customer contract, NDA, broker-carrier)
- Search by name or content
- Template status (Active, Draft, Archived)
- Version tracking

## Acceptance Criteria
- [ ] Contract templates list with filtering
- [ ] Template status management
- [ ] Version history visible
- [ ] Create new template action
- [ ] Archive/restore functionality
- [ ] Pagination and search

## Dependencies
