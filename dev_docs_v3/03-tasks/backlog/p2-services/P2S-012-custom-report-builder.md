# P2S-012: Custom Report Builder

**Priority:** P2
**Service:** Analytics
**Scope:** Self-service report builder with data source selection and export

## Current State
No reporting system exists.

## Requirements
- Data source selection (loads, carriers, invoices, etc.)
- Column picker and ordering
- Filter builder (date range, status, entity)
- Grouping and aggregation (sum, count, average)
- Chart type selection (bar, line, pie, table)
- Export to CSV, PDF, Excel
- Save report as template
- Schedule recurring reports

## Acceptance Criteria
- [ ] Data source selection
- [ ] Column and filter configuration
- [ ] Grouping and aggregation
- [ ] Multiple chart types
- [ ] Export to CSV/PDF/Excel
- [ ] Save as template
- [ ] Schedule recurring delivery

## Dependencies
- P2S-010 (Analytics dashboards)
