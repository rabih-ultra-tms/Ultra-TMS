# P2S-020: Scheduler/Cron Management

**Priority:** P2
**Service:** Scheduler
**Scope:** Admin UI for managing scheduled tasks and cron jobs

## Current State
No scheduler management UI exists. Backend may have scheduled tasks defined in code.

## Requirements
- List of all scheduled tasks with status
- Enable/disable individual tasks
- Cron expression editor with human-readable preview
- Execution history log per task
- Manual trigger button
- Next execution time display

## Acceptance Criteria
- [ ] Scheduled task list with status
- [ ] Enable/disable toggle
- [ ] Cron expression editor
- [ ] Execution history with success/failure
- [ ] Manual trigger button
- [ ] Next execution time shown

## Dependencies
- Backend scheduler module

M
