# P2S-010: Analytics Dashboards Builder

**Priority:** P2
**Service:** Analytics
**Scope:** Configurable analytics dashboards with drag-and-drop widgets

## Current State
No analytics dashboard exists. Individual stats cards exist on some list pages (carriers, load-history).

## Requirements
- Dashboard builder with drag-and-drop widget placement
- Pre-built widget library (charts, KPIs, tables)
- Data source selection per widget
- Date range selector (global and per-widget)
- Save and share dashboards
- Auto-refresh capability

## Acceptance Criteria
- [ ] Dashboard builder with grid layout
- [ ] Widget library with 10+ widget types
- [ ] Drag-and-drop placement and resizing
- [ ] Data source configuration per widget
- [ ] Dashboard save and share
- [ ] Auto-refresh on configurable interval

## Dependencies
- Analytics backend with aggregation queries
