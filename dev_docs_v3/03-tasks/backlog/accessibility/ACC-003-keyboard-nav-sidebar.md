# ACC-003: Keyboard Navigation for Sidebar

**Priority:** P1
**Service:** Frontend Accessibility
**Scope:** Keyboard-accessible sidebar navigation

## Current State
Sidebar links are likely tabbable via standard anchor elements. Sub-menu expand/collapse may not be keyboard accessible. No skip-to-content link exists.

## Requirements
- All sidebar links tabbable in logical order
- Sub-menu expand/collapse with Enter/Space
- Arrow keys to navigate between menu items
- Visual focus indicator on active item
- Skip-to-content link (see ACC-008)

## Acceptance Criteria
- [ ] All menu items tabbable
- [ ] Sub-menus toggle with Enter/Space
- [ ] Arrow key navigation works
- [ ] Focus indicator visible on all items
- [ ] Current page indicator distinguishable from focus

## Dependencies
- None

## Estimated Effort
M
