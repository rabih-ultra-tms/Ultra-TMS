# UX-008: Keyboard Shortcuts for Power Users

**Priority:** P2
**Service:** Frontend UX
**Scope:** Add keyboard shortcuts for common actions

## Current State
No keyboard shortcuts exist. Power users (dispatchers, accountants) would benefit from keyboard-driven navigation.

## Requirements
- Global shortcuts: Cmd/Ctrl+K for search, Cmd/Ctrl+N for new entity
- Navigation shortcuts: G+L for loads, G+C for carriers, G+O for orders
- Table shortcuts: J/K for row navigation, Enter to open, Delete to remove
- Help overlay showing all shortcuts (press ?)

## Acceptance Criteria
- [ ] Search shortcut (Cmd/Ctrl+K) opens command palette or search
- [ ] New entity shortcut works on list pages
- [ ] Navigation shortcuts implemented
- [ ] Help overlay lists all shortcuts
- [ ] Shortcuts don't conflict with browser defaults

## Dependencies
- None
