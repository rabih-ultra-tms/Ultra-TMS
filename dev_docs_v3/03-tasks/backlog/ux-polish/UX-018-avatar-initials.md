# UX-018: Avatar/Initials for User References

**Priority:** P2
**Service:** Frontend UX
**Scope:** Show avatar or initials for user references throughout the app

## Current State
User references (created by, assigned to, dispatcher, etc.) show raw text or IDs. No avatar or initials component exists.

## Requirements
- Create Avatar component with initials fallback
- Show avatars next to user references (assigned dispatcher, created by, etc.)
- Color-code initials based on user name hash
- Support profile image when available

## Acceptance Criteria
- [ ] Avatar component created with initials fallback
- [ ] Consistent color coding for initials
- [ ] Applied to user references across the app
- [ ] Profile image support when available
- [ ] Accessible (alt text, screen reader support)

## Dependencies
- None

## Estimated Effort
S
